# MVP 需求文档：图片背景去除工具

**版本**：v1.0  
**日期**：2026-03-19  
**状态**：初稿

---

## 1. 产品概述

| 项目 | 内容 |
|------|------|
| **产品名称** | 简图 - 背景去除工具 |
| **产品类型** | Web 在线工具 |
| **核心功能** | 一键去除图片背景，下载透明背景图 |
| **目标用户** | 电商卖家、设计师、自媒体创作者 |

---

## 2. MVP 范围

### 2.1 核心功能（必须做）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 支持点击/拖拽上传 PNG/JPG/WEBP | P0 |
| 背景去除 | 调用 Remove.bg API 处理图片 | P0 |
| 结果预览 | 同时展示原图和处理后的图片 | P0 |
| 图片下载 | 一键下载去除背景后的图片 | P0 |
| API 接入 | 使用 Cloudflare Workers 中转，隐藏 API Key | P0 |

### 2.2 扩展功能（暂不做）

- 批量处理
- 历史记录
- 积分/付费系统
- AI 扩图
- 移动端 App

---

## 3. 用户流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  上传图片   │ -> │  等待处理   │ -> │  预览结果   │ -> │  下载保存  │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

1. 用户打开网站
2. 点击上传区域或拖拽图片
3. 系统自动上传并处理（约 2-5 秒）
4. 显示原图和结果图对比
5. 用户点击下载保存

---

## 4. 技术方案

### 4.1 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端 | HTML + CSS + JS | 静态页面，无后端 |
| 中转 | Cloudflare Workers | 免费额度内无费用 |
| AI 服务 | Remove.bg API | $0.049/张 |
| 托管 | Cloudflare Pages | 免费托管 |

### 4.2 架构图

```
用户浏览器                    Cloudflare                    Remove.bg
    │                            │                             │
    │  ┌──────────────────┐     │     ┌─────────────────┐     │
    │─>│  前端页面         │────>│────>│  Worker 中转    │────>│
    │  │ (上传/预览/下载)  │     │     │  (隐藏 API Key) │     │
    │  └──────────────────┘     │     └─────────────────┘     │
    │                            │                             │
    <────────────────────────────│                             │
         返回透明背景图           <─────────────────────────────┘
```

### 4.3 成本估算

| 项目 | 免费额度 | 超出费用 |
|------|----------|----------|
| Cloudflare Workers | 10 万次/月 | $0.15/百万次 |
| Cloudflare Pages | 无限 | - |
| Remove.bg | 50 张/月 | $0.049/张 |

**MVP 阶段预估**：免费额度够用

## 6. 功能详细说明

### 6.1 图片上传

- 支持格式：PNG, JPG, WEBP
- 最大尺寸：10MB
- 交互方式：点击上传框 或 拖拽文件

### 6.2 背景去除

- 处理时间：2-5 秒
- 输出格式：PNG（透明背景）
- 自动处理：上传后立即开始

### 6.3 结果展示

- 双图对比：原图 vs 结果图
- 透明背景预览：棋盘格背景

### 6.4 下载功能

- 默认文件名：removed-bg.png
- 直接触发浏览器下载
